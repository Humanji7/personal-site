Продолжаем работу над личным сайтом как путешествием. Готовы начать кодить точку входа.

- `PROTOTYPE_SPHERE.md` — спецификация прототипа

## Что уже есть

### Закрыто:
- Философия и смысловое ядро
- Спецификация прототипа сферы

- План из 5 этапов

### Этап 1 (старт):
- Чёрный экран

- Частицы дышат (пульсация)

---

## Метод работы

1. Делаем один этап
2. Запускаем `npm run dev`
3. Смотрим результат
4. Я даю направление
5. Следующий этап или правка

**Кисточка, не чертёж.**

---

## Команда для старта

```
Начни с Этапа 1: создай проект с Vite + Three.js, сделай чёрный экран и дышащую сферу из частиц.
```

---

## Визуальные параметры

- Фон: #000000
- Частицы: #2D1B4E → #1A0F2E (грязно-фиолетовый)
- Количество: ~2000-5000
- Пульсация: ±5%, цикл 3-4 сек
