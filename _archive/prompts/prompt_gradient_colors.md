# Prompt: Градиентные цветовые переходы

## Контекст проекта

Интерактивная сфера с эмоциональными состояниями. Сейчас цвета меняются через `setColorTint()` и `setColorHue()`, но переходы резкие.

**Расположение:** `/Users/admin/projects/personal-site/prototype-sphere/`

## Текущая проблема

Цветовые переходы между состояниями:
- `PEACE` (фиолетовый #BB9FDF) 
- `TENSION` (затемнённый + холодный hue shift)
- `BLEEDING` (ember #E68060)
- `HEALING` (тёплый → нормальный)

Сейчас переходы линейные в RGB → **резкий скачок**, особенно между далёкими оттенками.

## Задача

Сделать цветовые переходы **градиентными через HSL**, чтобы:
1. Переход проходил через все промежуточные оттенки
2. Не было "прыжков" между несовместимыми цветами
3. Сохранялась эмоциональная выразительность каждого состояния

## Варианты реализации

| Вариант | Описание | Сложность |
|---------|----------|-----------|
| **A: HSL lerp в shader** | `mix()` в HSL-пространстве | Средняя |
| **B: Color palette uniform** | Массив цветов, интерполяция по uPhase | Средняя |
| **C: JS-side HSL → RGB** | Расчёт в JS, передача готового RGB | Низкая |

## Ключевые файлы

```
prototype-sphere/src/
├── ParticleSystem.js  # Fragment shader, uColorNormal/uColorTint
├── Sphere.js          # setColorTint(), setColorHue()
```
