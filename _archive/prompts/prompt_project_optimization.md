# ЗАДАЧА: Оптимизация структуры проекта для агентов

## Контекст
Проект SPHERE (интерактивная сфера на Three.js) вырос органически через серию сессий. 
Накопились промпт-файлы, дублирующая документация, и агенту сложно быстро сориентироваться.

**Цель:** Реорганизовать проект так, чтобы новый агент мог за 30 секунд понять архитектуру и начать работу.

---

## Твои задачи

### 1. Аудит файлов в корне
- Пройдись по всем `.md` файлам в `/Users/admin/projects/personal-site/`
- Определи статус каждого: `current`, `outdated`, `archive`
- Составь таблицу рекомендаций

### 2. Создай структуру папок
```
personal-site/
├── prototype-sphere/       # Исходники сферы (уже есть)
├── docs/                   # Актуальная документация
│   ├── ARCHITECTURE.md     # Карта модулей и их связей
│   ├── PHILOSOPHY.md       # Философия проекта
│   └── CHANGELOG.md        # История изменений
├── _archive/               # Устаревшие материалы
│   └── prompts/            # Выполненные промпты
└── README.md               # Точка входа для агентов
```

### 3. Создай ARCHITECTURE.md
- Диаграмма модулей (Mermaid)
- Описание каждого модуля (1-2 предложения)
- Ключевые точки расширения
- Конфигурационные параметры

### 4. Создай агент-дружелюбный README.md
Структура:
```markdown
# SPHERE Prototype

## Быстрый старт для агента
1. `cd prototype-sphere && npm run dev`
2. Архитектура: [docs/ARCHITECTURE.md]
3. Философия: [docs/PHILOSOPHY.md]

## Текущее состояние
- Версия: 1.0 (Stage 6 завершён)
- Деплой: humanji.dev

## Ключевые файлы
| Файл | Назначение |
|------|------------|
| Sphere.js | Эмоциональный стейт-машин |
| ParticleSystem.js | GPU частицы и шейдеры |
| ... | ... |
```

### 5. Архивируй выполненные промпты
- Перенеси все `prompt_*.md` в `_archive/prompts/`
- Оставь только активные задачи

---

## Ограничения
- Не трогай `prototype-sphere/src/` — код стабилен
- Сохрани `PHILOSOPHY.md` (перенеси в `docs/`)
- Не удаляй — только архивируй

## Результат
После выполнения агент должен видеть чистый корень с README.md, 
и находить всё нужное за 1-2 клика.
