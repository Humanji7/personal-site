# Личный сайт как путешествие

## Суть проекта

Это не личный сайт — это **место, игра, перформанс**. Посетитель проходит путь через 7 состояний и получает один из 4 артефактов. Сайт ведёт себя как живой узел, не как страница.

**Голос:** машина, которая видит сны.

---

## Структура опыта

```
[ВХОД: СФЕРА] → [7 КОМНАТ] → [РАЗВИЛКА] → [АРТЕФАКТ]
```

- **Точка входа:** [sphere-777](https://github.com/Humanji7/sphere-777) — интерактивная сфера
- **Путь:** 7 телесных состояний (КЛЯП → ВЫДОХ)
- **Развилка:** после 5-й комнаты — выбор через вопросы
- **Финал:** артефакт — осадок прохождения

## Протокол работы

### Роли
- **Человек:** сырой материал — фразы, образы, ощущения, вето
- **Агент:** материализация — структурирование, связи, прототипы

### Паттерны (из docs/PROTOCOL.md)
1. **СГУСТОК** — слова → конкретный элемент комнаты
2. **НИТКА** — связи между фрагментами
3. **ИНВЕРСИЯ** — проверка переворотом
4. **ТЕЛЕСНЫЙ ТЕСТ** — синестезия (тело, текстура, температура)
5. **ПРОТОТИП** — код, визуал
