## Результат

**V7** — финальная версия фона комнаты КЛЯП (Canvas-based).

### Что включено

| Элемент | Описание |
|---------|----------|
| **Органические blob-ы** | 6 дышащих масс с noise-деформацией контура |
| **Вены** | 12 волнистых линий с пульсацией толщины |
| **Реакция на курсор** | Blob-ы притягиваются к мышке (вязкая задержка) |
| **Виньетка** | Пульсирующее давление с краёв (CSS layer) |
| **Tremor** | Весь контент дрожит каждые 3s |
| **Кастомный cursor-glow** | Мягкое пятно вместо стандартного курсора |
| **Текст** | Интегрирован с дыханием фона |

### Что убрано

- ❌ `swallowWaves` (кольцевая пульсация) — лишний элемент, отвлекал
