# Project Brief

Интерактивный перформанс — "место, игра" — где посетитель проходит путь через 7 телесных состояний и получает артефакт.

---

## Суть в одном предложении

GPU-рендеринг 100k частиц, которые ведут себя как **ветер+вода** под влиянием курсора-пастуха, с bloom/tonemap пайплайном и системой качества.

---

## Goals

1. **Ощущение живого места** — не демо шейдера, а пространство
2. **Визуальный язык** — частицы как "расплавленный свет", не песок
3. **Производительность** — 60 FPS на современных устройствах, graceful degradation
4. **Обратимый скролл** — путешествие туда-обратно без артефактов

---

## Non-Goals

- Не SPA с роутингом
- Не многопользовательский опыт
- Не музыкальный визуализатор (хотя аудио возможно позже)

---

## Ключевые метафоры

| Метафора | Применение |
|----------|------------|
| **Курсор-пастух** | Ведёт, не толкает. Частицы следуют за "тенью" позади курсора |
| **Ветер+вода** | curl noise + ribbon path вместо взрыва/притяжения |
| **Расплавленный свет** | Мягкие ореолы, не точки |
| **Тёплый шиммер** | Пустота не мёртвая — мерцает в негативном пространстве |

---

## Текущая стадия

**Технический фундамент готов:**
- GPGPU particle system (100k)
- PostFX pipeline (bloom + ACES tonemap)
- Quality tiers (auto-scaling)
- Scroll orchestration

**Открытый фронт:**
- Визуальное качество частиц ("бедно и дёшево")
- Поиск образа-метафоры для перехвата дыхания
