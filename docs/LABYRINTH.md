## Философское ядро

**Юзер доносит идею себе сам.**

Автор не даёт смысл — автор создаёт условия, при которых юзер *порождает* смысл.

**Архитектура:**
- ИИ = мета-мозг (тысячи историй, сотни артефактов)
- Юзер = триггер генерации («трёт лампу джина»)
- Артефакт = не готовый объект, а результат генерации под конкретного юзера

**Сигналы лампы (что собираем):**
- Текст (вводит / надиктовывает)
- Цвета (выбор)
- Движение (траектории на картах/канвасе)
- Атомарные значения (гейм-комбинации мелких выборов)
- Логи пребывания (время, паузы, ритм, путь)
- «Ветер» (неизмеряемое — само пришествие помыслов)

**Техническая архитектура:**
- Сигналы → классификатор (200 бакетов поведения)
- Каждый бакет → жёсткий системный промпт
- Промпт → быстрая модель → генерация (текст / музыка / визуал)
- Гармоничное переключение модальностей в потоке

| Что происходит | Что юзер думает |
|----------------|-----------------|
| Читает нарратив | «Мне рассказывают» |
| Слышит музыку | «Атмосфера» |
| Выбирает путь | «Я навигирую» |
| Генерирует итог | **«Я нашёл»** |

**Парадокс:** юзер создаёт, но думает, что открывает.

**Следствия:**
- Не задавать вопросы явно — завлекать в игру
- Юзер не должен понять, что произошло
- Артефакт = память о соприкосновении
- Даже «потратил время зря» = валидный результат

**Жанр:** отсутствие жанра. Структура хаоса. Сложные инженерные и визуальные завихрения — зритель просто выбирает себя.

**Финал (синестезия):** ощущение над головой, трепетное сверление — божественное и обыкновенное одновременно.

## Как это работает

**Твоя роль:** приносить сырое — фразы, ощущения, противоречия, «не знаю как сказать».

**Моя роль:** материализовать через паттерны:

| Паттерн | Что делает | Когда |
|---------|------------|-------|
| **СГУСТОК** | Твои слова → конкретный элемент комнаты | Есть образ/ощущение |
| **НИТКА** | Связываю разрозненные идеи | Есть 2+ фрагмента |
| **ИНВЕРСИЯ** | Переворачиваю для проверки | Что-то слишком гладко |
| **ТЕЛЕСНЫЙ ТЕСТ** | Где в теле? Какая текстура? | Нужна глубина |
| **ПРОТОТИП** | Пишу код, показываю | Пора трогать руками |
