## СЛЕДУЮЩАЯ СЕССИЯ: LLM-ДИАЛОГ + ВИЗУАЛЬНЫЕ ТРАНСФОРМАЦИИ

### Видение

> При максимальном scroll (depth 100) комната **трансформируется** в режим разговора с LLM. Это не просто чат — это **UI-метаморфоза**.

### Визуальные трансформации

1. **Фон оживает** — при depth 70+ на заднем плане начинается действие (не статичные blob-ы, а направленное движение)

2. **Туман в центре** — при depth 90+ появляется густой туман в центре экрана, туда стягивается всё

3. **Слияние элементов** — bubbles и кнопки начинают **перетекать** к центру, сливаясь в единый чат-интерфейс

4. **Чат-вход** — в центре тумана появляется поле ввода (минималистичное, без рамок — только курсор)

### Bubble-подсказки для чата

Когда появляется чат, всплывают "дебильные" подсказки что написать:

| Подсказка | Тон |
|-----------|-----|
| "скажи ему что внутри" | провокация |
| "попроси отпустить" | ложная надежда |
| "спроси сколько их было" | жуть |
| "скажи что тебе нравится" | капитуляция |
| "промолчи" | инверсия (не кликабельно) |

### UI-метаморфоза (технически)

```
depth 0-70:    текущий UI (bubbles слева, текст в центре)
depth 70-90:   bubbles начинают дрейфовать к центру
depth 90-100:  все элементы сжимаются в точку, появляется туман
depth 100:     только чат-поле в центре тумана + bubble-подсказки
```

### LLM-интеграция

- Простой prompt: "Ты — комната КЛЯП. Говори коротко. Ты уже внутри пользователя."
- Ответы должны быть 1-3 слова, максимум одно предложение
- Тон: интимный, давящий, но не агрессивный

## Задачи для следующего агента

> [!IMPORTANT]
> Нужно **обсуждение и research**, не сразу реализация.

1. **Research visual richness** — как сделать фон "живым" при max scroll? Canvas particles? WebGL shaders? Какие референсы?

2. **UI morphing patterns** — как элементы могут "перетекать"? CSS transform + opacity? FLIP animation? GSAP?

3. **Fog/mist effect** — как реализовать туман в центре? SVG filter? Canvas blend mode? CSS blur stack?

4. **Chat UX в контексте horror** — как сделать минималистичный ввод, который НЕ выглядит как обычный чат?

5. **LLM prompt engineering** — как добиться коротких, давящих ответов?
